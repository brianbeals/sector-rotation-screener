# Weekly runs

Auto-generated every Sunday afternoon ET by GitHub Actions. The screener pulls fresh data from yfinance and FRED, scores the 11 SPDR sector ETFs, runs the 15-year backtest, and asks Claude for a short commentary on what the screen said.

## Latest

The first scheduled run has not happened yet. After the first run, this section will link to:

- `latest/dashboard.html` — the most recent screen output
- `latest/summary.md` — Claude's plain-language reading

You can trigger the first run manually from the [Actions tab](../../actions/workflows/weekly.yml) once the `FRED_API_KEY` and `ANTHROPIC_API_KEY` secrets are set.
